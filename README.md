# luckysimulator-Patch

饥荒联机版模组 **luckysimulator-Patch**，用于为依赖模组 **欧皇模拟器** 增加一些便捷补丁功能。

## 创意工坊

Steam 创意工坊订阅链接：[luckysimulator-Patch](https://steamcommunity.com/sharedfiles/filedetails/?id=3656655913)
欧皇模拟器订阅链接：[欧皇模拟器](https://steamcommunity.com/sharedfiles/filedetails/?id=3273001012)

## 功能简介

- 增加金币兑换抽奖机相关物品、附魔相关物品和海钓浮漂。
- 可配置开局礼包金币数量。
- 可配置抽奖机商店是否显示全部物品。
- 可配置商店价格倍率和额外金币爆率倍率。
- 可选击杀掉落金币直接存入玩家账户。
- 支持使用 `#ql` 清理附近掉落物，并按价值回收为金币。

## 使用说明

1. 订阅并启用依赖模组 **欧皇模拟器**。
2. 订阅并启用本补丁模组 **luckysimulator-Patch**。
3. 在世界模组配置中按需调整兑换、商店、金币和回收相关选项。
